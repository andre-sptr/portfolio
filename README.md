# Portfolio

Personal portfolio website for Andre Saputra, focused on network, IoT, telecom automation, and web development projects.

## Features

- Project showcase for selected GitHub repositories.
- Personal profile and contact sections.
- Modern React UI with reusable components.
- Production server scripts for deployment workflows.

## Tech Stack

- React
- Vite
- TypeScript
- Tailwind CSS
- Radix UI / shadcn-style components
- Node.js server scripts

## Getting Started

```bash
npm install
npm run dev
```

Build for production:

```bash
npm run build
```
